# Outreach Follow-Up Cadence — Gemini-HMS

## Rules of Engagement

- **Personalize every message**. Never use a generic blast.
- **Maximum 3 attempts** before moving to "Paused" unless prospect engages.
- **Respect opt-out immediately**. Remove from all lists.
- **Minimum 3 business days** between attempts (except demo confirmation).
- **No phone calls unless prospect provides phone number**.

## Cadence Steps

### Step 1: Initial Outreach (Day 0)

| Item | Detail |
| :--- | :--- |
| **Purpose** | Introduction and demo invitation. |
| **Channel** | Email or LinkedIn (choose based on context). |
| **Template** | `client-demo-invitation.md` (Option 1 for email, Option 2 for LinkedIn). |
| **Attachments** | `client-summary.md` (optional — link instead if possible). |
| **Action** | Request 30-min demo or introductory call. |

### Step 2: First Follow-Up (Day 4)

| Item | Detail |
| :--- | :--- |
| **Purpose** | Gentle reminder. Assume they are busy. |
| **Tone** | Helpful, not pushy. |
| **Content** | Short: "Following up on my previous message. Happy to find a time that works for you." |
| **Action** | Offer 2–3 specific time slots. |

### Step 3: Second Follow-Up (Day 10)

| Item | Detail |
| :--- | :--- |
| **Purpose** | One last attempt before pausing. |
| **Content** | Brief value reminder: "We've been helping clinics improve lab-to-dispense audit trails. Would a 15-min call work?" |
| **Action** | If no response → move to Paused / Closed Lost. |

### Step 4: Demo Confirmation (Scheduled Date)

| Item | Detail |
| :--- | :--- |
| **Purpose** | Confirm meeting, share logistics. |
| **Content** | Meeting link, duration (30 min), synthetic data reminder, agenda summary. |
| **Action** | Ask prospect to confirm attendance. |

### Step 5: Post-Demo Thank You (Day of Demo + 1)

| Item | Detail |
| :--- | :--- |
| **Purpose** | Thank prospect, recap key points, next steps. |
| **Template** | `client-follow-up-templates.md` (Post-Demo Thank You). |
| **Action** | Propose staging proposal or technical deep dive. |

### Step 6: Staging Proposal Follow-Up (If Proposal Sent, Day +5)

| Item | Detail |
| :--- | :--- |
| **Purpose** | Check if they reviewed the staging proposal. |
| **Content** | "Wanted to see if you had any questions on the staging scope and costs." |
| **Action** | Offer a call to walk through the `staging-proposal-checklist.md`. |

### Step 7: Nurture Follow-Up (Monthly)

| Item | Detail |
| :--- | :--- |
| **Purpose** | Stay on radar for prospects who paused or had long timelines. |
| **Content** | Light touch: share a relevant insight, product update, or industry article. No sales pitch. |
| **Action** | Only if prospect agreed to nurture. |

### Step 8: Stop / Opt-Out

| Trigger | Action |
| :--- | :--- |
| Prospect says "not interested" | Mark "Closed Lost". Do not contact again. |
| No response after 3 attempts | Move to "Paused". Optionally revisit in 3 months. |
| Prospect requests removal | Remove from all lists immediately. Log opt-out. |
| Disqualifying signal detected | Move to "Closed Lost". Do not re-engage. |
