# Phase 21: Real Contact Verification Checklist

**Mandatory**: Before any message is sent to a real prospective client, the operator must verify the contact channel using this checklist.

## 1. Official Source Verification
- [ ] **Official Website**: Contact info is listed on the organization's primary domain (e.g., `organization.com/contact`).
- [ ] **Public Directory**: Role-based inbox (e.g., `info@`, `admin@`) is confirmed in a public directory or LinkedIn company page.
- [ ] **Source URL Recorded**: The exact URL where the contact info was found is stored in the tracker.

## 2. Channel Integrity
- [ ] **No Personal Emails**: Verified that the address is role-based or organization-managed, not a personal account (`@gmail.com`, `@yahoo.com`, etc.).
- [ ] **No Personal Cell Numbers**: Verified that any phone number listed is an official office line.
- [ ] **No Scraped Data**: Confirmed contact was found via manual research, not automated scraping tools.

## 3. Deliverability & Ethics
- [ ] **Working Channel**: Verified that the contact form or inbox is active (no bounce-back in prior tests).
- [ ] **No Demo Data**: Confirmed the address is not a placeholder, test account, or demo domain.
- [ ] **Ethical Access**: No login-bypass, personal social media DM, or private profile harvesting was used.

## 4. Final Verification Logic
- [ ] **Manual Review**: A second team member has verified the contact channel for at least 1 in 5 prospects.
- [ ] **Owner Approval**: Final send list approved by the Project Owner AFTER real contact verification.
