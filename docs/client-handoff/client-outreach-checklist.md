# Gemini-HMS Client Outreach Checklist

## 1. Ideal Client Profile (ICP)
- [ ] **Type**: Multi-branch clinic networks or diagnostic centers.
- [ ] **Needs**: Strict clinical audit trails, role-based portal isolation, and integrated pharmacy/lab workflows.
- [ ] **Infrastructure**: Willing to fund/provision their own Google Cloud or AWS environment.

## 2. Outreach Handoff Package (What to Send)
- [ ] `client-summary.md`: Short executive overview.
- [ ] `client-demo-invitation.md`: Formal demo request message.
- [ ] `product-overview.md`: High-level value proposition.
- [ ] `known-limitations-and-next-steps.md`: Honest status on cloud/compliance.

## 3. Pre-Call Discovery Checklist
- [ ] Research clinic size and number of active clinicians.
- [ ] Identify current system pain points (e.g., lack of LIS/Pharmacy integration).
- [ ] Confirm stakeholder roles attending the demo (Medical Director, IT Head, Operations).
