## tracker Management Rules
1. **Minimize Retention**: Store only what is necessary to move to the Staging phase.
2. **Abstract Security**: If a client asks a specific security question, note the topic (e.g., "MFA implementation") but do not store the client's internal network details.
3. **Trigger Review**: Mark `Staging Ready?` as YES only if all criteria in `phase-21-staging-trigger-plan.md` are documented as met.
