# Phase 21: Opt-Out / Do-Not-Contact Procedure

Establishing a rigorous process for respecting privacy and maintaining ethical outreach boundaries.

## 1. What Counts as an Opt-Out
- Any response containing: "Stop," "Unsubscribe," "Not interested," "Remove me," "Do not contact."
- Any formal privacy request or complaint.
- Negative social media feedback regarding outreach.

## 2. Immediate Actions
1. **Cease Communication**: Stop all active message threads immediately.
2. **Flag Tracker**: Update the `phase-21-outreach-tracker-seed.md` (or production CRM) with:
   - `Opt-Out`: [X]
   - `Status`: STOP
3. **Internal Sync**: Notify any team members involved in that prospect relationship.

## 3. No-Follow-up Rule
- **ZERO** further contact is allowed after an opt-out is received. 
- Exception: A single, short confirmation message (e.g., "Understood. You have been removed from our list. Best regards.") is permitted but not required.

## 4. How to Handle Ambiguous "Not Now"
- If a prospect says "Check back in 6 months," do NOT mark as Opt-Out.
- Mark as `Status`: Nurture.
- Schedule a reminder for 6 months.

## 5. Handling Complaints
- If a prospect complains about outreach frequency or method:
  1. Acknowledge and apologize sincerely.
  2. Explain that we prioritize ethical outreach and have marked them for permanent removal.
  3. Escalate to the Project Owner for a process review.

## 6. Avoiding Unnecessary Data Storage
- Do not store the reason for opt-out if it contains sensitive personal details.
- Store only the organization name and the "Do Not Contact" flag.

## 7. Audit Note Format
Use this format in the tracker notes:
`[Date] Opt-out requested via [Channel]. Permanent stop applied.`
*(Do not include the specific text of a angry or sensitive message)*.
