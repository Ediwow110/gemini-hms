# Outreach Tracker Template — Gemini-HMS

## Anti-Spam Rules (Must Read Before Using)

1. **Personalize every message**. Do not use mail-merge blasts.
2. **No more than 3 outreach attempts per prospect** unless they respond positively.
3. **Respect opt-out immediately**. Remove from tracker and do not re-contact.
4. **No scraping**. Only use public business contact info or inbound leads.
5. **Do not share this tracker externally** — internal sales tool only.

## Columns

| # | Field | Description | Example |
| :--- | :--- | :--- | :--- |
| 1 | **Prospect** | Organization name | "City Lab Network" |
| 2 | **Facility Type** | Clinic chain, lab, outpatient, hospital | "Diagnostic Lab Center" |
| 3 | **Contact Channel** | Email, LinkedIn, web form, referral | "LinkedIn" |
| 4 | **Contact Role** | Role-based title (not personal data) | "IT Director" |
| 5 | **Outreach Date** | Date of first contact | "2026-05-26" |
| 6 | **Message Version** | Which template / variant used | "Demo Invitation v1" |
| 7 | **Response** | Summary of reply or note of no reply | "Interested, asked about HIPAA status" |
| 8 | **Demo Scheduled?** | Yes / No / Pending | "Pending" |
| 9 | **Qualification Status** | High / Medium / Low / Stop | "Medium" |
| 10 | **Budget / Staging Fit** | Yes / No / Unknown | "Unknown — need to ask" |
| 11 | **Compliance Concerns** | HIPAA cert, data residency, etc. | "Asked about SOC2 timeline" |
| 12 | **Next Follow-Up Date** | When to contact next | "2026-06-02" |
| 13 | **Notes** | Free text context | "Prefers email. Wants technical deep dive before demo." |
| 14 | **Status** | See statuses below | "Interested" |

## Status Values

| Status | Meaning |
| :--- | :--- |
| **Not Contacted** | Research complete, outreach not yet sent. |
| **Contacted** | First outreach sent. Awaiting response. |
| **Interested** | Responded positively. Needs further conversation. |
| **Demo Scheduled** | Demo date confirmed. |
| **Demo Completed** | Demo delivered. |
| **Staging Proposal Sent** | Formal staging proposal shared. |
| **Closed Lost** | Prospect declined or disqualified. |
| **Paused** | Prospect interested but timing wrong. Revisit in 3 months. |

## Disqualification Quick-Check

If ANY of these are true, set Status to "Closed Lost":
- Prospect demands production deployment immediately.
- Prospect requires HIPAA/SOC2 certification proof before demo.
- Prospect unwilling to fund staging (~$50–150/mo).
- Prospect needs eRx / insurance clearinghouse / ONC certification.
- Prospect wants demo with real patient data.
- No response after 3 personalized attempts.

## Tracker Format Suggestion

Maintain as a spreadsheet (Google Sheets, Excel, or Airtable) with the columns above. Update within 24 hours of any contact or response.
