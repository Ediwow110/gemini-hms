# Phase 21: Batch 1 Outreach Prep Checklist

Mandatory verification for every message sent in the first outreach batch.

## 1. Ethics & Privacy
- [ ] **Public Source**: Verified organization info came from an official website or public listing.
- [ ] **No Private Data**: Confirmed NO personal cell numbers or personal emails are in the tracker or message.
- [ ] **Official Channel**: Using info@, admin@, or official web forms only.

## 2. Technical Truthfulness
- [ ] **"Local Green"**: Message explicitly frames Gemini-HMS as a "locally verified development baseline."
- [ ] **"Synthetic Only"**: Message explicitly states that demos use "strictly synthetic data."
- [ ] **"No Certification"**: Confirmed NO claims of active HIPAA or SOC2 certification are made.
- [ ] **"Not Live"**: Confirmed NO claims that the system is currently live on the cloud are made.

## 3. Personalization Quality
- [ ] **Truthful Familiarity**: Only said "I've been following..." if a specific news/PR item was found.
- [ ] **Document Check**: Only said "I've attached..." if the document is actually included in the message/form.
- [ ] **Role Fit**: Confirmed the target role is relevant to clinical or IT decision-making.

## 4. Governance & Closing
- [ ] **Opt-Out Clause**: Included a respectful close (e.g., "Please let me know if you would prefer not to receive further updates").
- [ ] **Tracker Sync**: Prospect status set to `RESEARCHED` or `READY` in the outreach tracker.
- [ ] **Approval**: No message sent without Batch 1 Send Review sign-off.

## 5. Decision Thresholds
- [ ] Stop if any uncertainty exists regarding the prospect's public status.
- [ ] Stop if the outreach channel requires a personal social login/bypass.
