1.  **Public Data Only**: Only include information found on official websites or public professional profiles.
2.  **Opt-Out Enforcement**: If "Opt-Out" column is checked, all communication must cease immediately.
3.  **No PHI**: Never use this tracker to store patient data or clinical secrets.
4.  **Verification**: Update the "Fit Score" using the criteria in `phase-21-prospect-scoring-matrix.md`.
